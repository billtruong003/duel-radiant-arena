## Root cause #1 — Server: `autoDispose=true` trên room rỗng

Khi bot pre-create room qua `/admin/create-room` (không có player nào join trước), Colyseus 0.15 thấy room có:
- 0 client
- 0 seat reservation
- `autoDispose=true`

→ room **tự dispose ngay sau onCreate**. matchMaker trả `code 4212 "room not found"` cho join sau đó. Unity SDK surface thành **"seat reservation expired"** (misleading).

### Đã fix (committed)

Repo: `D:\Project\radiant-bot`  
Commit: `9698874 fix(arena-server/Lát-D.U3b): autoDispose=false + join-deadline timer`

File: `arena-server/src/rooms/DuelRoom.ts`
- `this.autoDispose = false`
- `_joinDeadlineTimer` mới: nếu hết `join_deadline_at` mà chưa có ai join → dispose room (tránh leak slot)
- Cleanup trong `clearAllTimers()`

### Verify khi nào fix work
```bash
cd D:\Project\radiant-bot\arena-server
npm run dev          # terminal 1
# terminal 2:
npm run playtest -- --session=test1
# Lấy roomId từ output, rồi:
curl -X POST http://localhost:2567/matchmake/joinById/<ROOM_ID> \
  -H "Content-Type: application/json" -d '{"token":"dummy"}'
# Phải trả { room: {...}, sessionId: "..." } chứ KHÔNG phải code 4212
```

Mình đã test trực tiếp bằng `colyseus.js` (node script) — connect được, đọc được `phase=lobby`. **Server-side fix đã work xong**.

## Root cause #2 — Unity Colyseus SDK 0.17.16 ↔ server 0.15.x

Sau khi fix server, Unity **vẫn fail "seat reservation expired"** — nhưng bằng `colyseus.js` 0.15.28 thì connect được.

→ Phát hiện Unity manifest pin SDK qua URL floating:
```json
"io.colyseus.sdk": "https://github.com/colyseus/colyseus-unity-sdk.git?path=Assets/Colyseus"
```
URL không có `#tag/branch` → Unity pull **master** → master hiện đang ở **0.17.16**.

Server đang ở `@colyseus/core@^0.15.0`. Protocol handshake + serializer format khác giữa minor.

### Đã fix (committed)

Repo: `D:\Project\duel-radiant-arena`  
Commit: `7bf5fd5 fix(arena-unity/Lát-D.U3b): pin Colyseus SDK to 0.15 branch`

File: `Packages/manifest.json`
```json
"io.colyseus.sdk": "https://github.com/colyseus/colyseus-unity-sdk.git?path=Assets/Colyseus#0.15"
```

**CHƯA RE-RESOLVED** trên máy nhà vì Unity đang chạy với SDK cũ trong `Library/PackageCache/`. Cần đóng Unity → mở lại → Package Manager sẽ tự fetch branch `0.15`.
